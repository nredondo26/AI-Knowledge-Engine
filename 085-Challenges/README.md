# 085-Challenges: Desafíos Técnicos

## Descripción del dominio

Los desafíos técnicos son competiciones y actividades que ponen a prueba habilidades de programación, seguridad, análisis y resolución de problemas en escenarios complejos. Incluyen hackathons, coding competitions estilo ICPC/Codeforces, CTFs (Capture The Flag), bug bounties y retos por niveles de dificultad. Este módulo recopila plataformas, eventos y recursos para participar y mejorar.

## Conceptos clave

- **CTF (Capture The Flag)**: Competición de ciberseguridad donde se encuentran "banderas" ocultas en sistemas vulnerables
- **Hackathon**: Evento intensivo de desarrollo donde equipos crean prototipos en 24-48 horas
- **Bug Bounty**: Programa que recompensa económicamente por encontrar vulnerabilidades en software
- **Coding Competition**: Competencia algorítmica con problemas de dificultad creciente (Codeforces, ICPC)
- **Jeopardy CTF**: Formato de CTF con categorías (web, crypto, pwn, reverse, forensics)
- **Attack-Defense CTF**: Formato donde equipos atacan y defienden infraestructura en tiempo real
- **Speed Programming**: Resolver la mayor cantidad de problemas en el menor tiempo posible
- **Scoreboard**: Tabla de clasificación en tiempo real durante la competencia
- **Upsolving**: Resolver problemas después del concurso para aprender de las soluciones
- **Team Strategy**: Distribución de roles y estrategia de resolución en equipo

## Hoja de ruta

**Nivel Principiante:**
1. Participar en LeetCode Biweekly Contest (resolver 1-2 problemas)
2. Completar challenges introductorios de PicoCTF
3. Unirse a un hackathon local o virtual tipo DevPost
4. Resolver problemas de dificultad A en Codeforces

**Nivel Intermedio:**
1. Mantener rating 1600+ en Codeforces (Specialist)
2. Participar en CTF de formato Jeopardy con equipo
3. Reportar vulnerabilidades en programas públicos de bug bounty
4. Competir en Kaggle competitions con equipo

**Nivel Avanzado:**
1. Alcanzar rating 2100+ en Codeforces (Candidate Master / Master)
2. Clasificar a finales de ICPC o Google Code Jam
3. Ganar recompensas significativas en HackerOne
4. Organizar un CTF o hackathon con desafíos propios
