# Core

## Descripción
Fundamentos y principios básicos de la computación y el desarrollo de software. Núcleo del conocimiento del que dependen todos los demás módulos.

## Conceptos clave
- Sistemas numéricos (binario, hexadecimal)
- Estructuras de datos fundamentales
- Algoritmos básicos
- Complejidad computacional (Big O)
- Memoria y almacenamiento
- Concurrencia y paralelismo
- Paradigmas de programación
