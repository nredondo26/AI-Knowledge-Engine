## Propósito

Este repositorio funciona como un **Knowledge Engine** consumible por:

- **Humanos**: Como referencia técnica estructurada
- **LLMs**: Como contexto RAG para respuestas precisas
- **Agentes**: Como base de conocimiento para tareas autónomas
- **MCP Servers**: Como fuente de datos estructurados

## Formato de contenido

Cada directorio contiene archivos en Markdown (`.md`) con estructura consistente:

