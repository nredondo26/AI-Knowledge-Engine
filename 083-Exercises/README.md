# 083-Exercises: Ejercicios Prácticos

## Descripción del dominio

Los ejercicios prácticos son fragmentos delimitados de código o problemas diseñados para reforzar conceptos técnicos específicos. Este módulo agrupa ejercicios por tecnología y nivel de dificultad, incluyendo katas (inspirados en Code Wars), coding challenges, ejercicios de algoritmos y estructuras de datos, y problemas resueltos con explicaciones paso a paso.

## Conceptos clave

- **Kata**: Ejercicio corto y repetible para practicar una habilidad concreta (concepto tomado del karate)
- **Coding Challenge**: Problema algorítmico o de diseño que debe resolverse en un tiempo limitado
- **Test-Driven Learning**: Aprendizaje mediante escritura de tests que validan la solución
- **REPL-based Exercise**: Ejercicio interactivo ejecutable en consola o notebook
- **Code Golf**: Práctica de escribir la solución más corta posible en número de caracteres
- **Refactoring Exercise**: Práctica de mejorar código existente sin cambiar su comportamiento
- **Peer Review**: Evaluación cruzada de soluciones entre pares
- **Time-boxed Exercise**: Ejercicio con límite de tiempo para simular presión real
- **Step-by-Step Solution**: Solución desglosada en pasos con explicación de cada decisión
- **Multiple Solutions**: Presentación de diferentes enfoques para un mismo problema

## Tecnologías principales

| Herramienta | Propósito |
|-------------|-----------|
| LeetCode | Plataforma de coding challenges con soporte para 14+ lenguajes |
| Codewars | Katas de programación con gamificación y rankings |
| HackerRank | Challenges por dominio (algoritmos, SQL, shell) |
| Exercism | Ejercicios gratuitos con mentores y tracks por lenguaje |

| Study Plan (LeetCode) | Rutas temáticas de 30/60/90 días |

## Hoja de ruta

**Nivel Principiante:**
1. Resolver katas básicas en Codewars (niveles 8-7 kyu)
2. Completar ejercicios de sintaxis en Exercism para el lenguaje elegido
3. Practicar manipulación de arrays y strings
4. Resolver los primeros 10 problemas de Project Euler

**Nivel Intermedio:**
1. Completar LeetCode Blind 75 (75 problemas esenciales)
2. Implementar estructuras de datos desde cero (linked list, tree, hash map)
3. Resolver katas de nivel 5-4 kyu en Codewars
4. Participar en los primeros días de Advent of Code

**Nivel Avanzado:**
1. Resolver problemas de dynamic programming y graph algorithms
2. Competir en contests semanales de LeetCode / Codeforces
3. Resolver katas de nivel 3-1 kyu en Codewars
4. Crear ejercicios propios con tests automatizados usando Vitest o Pytest
