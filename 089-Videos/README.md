# 089-Videos: Videos y Conferencias

## Descripción del dominio

Los recursos audiovisuales son fundamentales para el aprendizaje técnico, combinando explicaciones visuales, demostraciones en vivo y contenido actualizado de primera mano. Este módulo cataloga canales de YouTube, conferencias grabadas (talks técnicos), cursos en video, screencasts y listas de reproducción organizadas por tecnología y nivel. Incluye tanto contenido gratuito como plataformas de suscripción.

## Conceptos clave

- **Conference Talk**: Charla grabada en conferencia técnica (keynote, workshop, breakout session)
- **Screencast**: Grabación de pantalla narrada que muestra un proceso o tutorial paso a paso
- **Video Course**: Serie estructurada de videos que cubren un temario completo
- **Live Coding**: Transmisión en vivo o grabada de programación en tiempo real
- **Tech Channel**: Canal de YouTube especializado en contenido técnico
- **Playlist**: Lista de reproducción temática que agrupa videos relacionados
- **Workshop Recording**: Taller interactivo grabado durante un evento
- **Panel Discussion**: Debate entre expertos sobre un tema técnico
- **Crash Course**: Curso intensivo de corta duración que cubre lo esencial
- **Q&A Session**: Sesión de preguntas y respuestas grabada después de una charla

## Tecnologías principales

| Recurso | Propósito |
|---------|-----------|
| YouTube | Plataforma principal de videos técnicos y canales especializados |
| Pluralsight | Cursos en video profesionales con assessments y paths |
| Udemy | Cursos en video comerciales con certificados |
| Coursera / edX | Cursos universitarios en video con assignments |
| Frontend Masters | Cursos profundos de frontend y full-stack |
| egghead.io | Tutoriales cortos y enfocados en tecnologías web |
| FreeCodeCamp | Canal YouTube con cursos completos gratuitos |
| Confreaks | Grabaciones de conferencias técnicas (RailsConf, RubyConf) |
| InfoQ | Talks técnicos con transcripts y presentaciones |
| Twitch | Live coding y streams técnicos en vivo |

## Hoja de ruta

**Nivel Principiante:**
1. Explorar canales recomendados (Fireship, Traversy Media, The Net Ninja)
2. Seguir un crash course completo de HTML/CSS/JS (4-6 horas)
3. Ver playlist de introducción a Git y GitHub
4. Asistir a un live coding en Twitch/YouTube para ver flujo real

**Nivel Intermedio:**
1. Ver grabaciones de conferencias relevantes (React Conf, KubeCon, PyCon)
2. Completar curso especializado en Udemy/Pluralsight sobre tecnología elegida
3. Seguir screencast de debugging de problemas reales
4. Ver y analizar arquitectura de sistemas en videos de "System Design Interview"

**Nivel Avanzado:**
1. Asistir virtualmente a conferencias internacionales y ver talks en directo
2. Crear contenido técnico propio (canal de YouTube, screencasts, workshops)
3. Formar grupo de visualización de conferencias con discusión post-talk
4. Contribuir a conferencias como speaker o revisor de propuestas
