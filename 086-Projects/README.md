# 086-Projects: Proyectos Completos

## Descripción del dominio

Los proyectos completos son implementaciones integrales que combinan múltiples tecnologías y disciplinas para resolver un problema real o simulado. A diferencia de los ejercicios o laboratorios, los proyectos abarcan frontend, backend, base de datos, despliegue, testing y documentación. Este módulo contiene proyectos de ejemplo, templates, aplicaciones demo y casos de estudio completos que sirven como referencia arquitectónica.

## Conceptos clave

- **Full-Stack**: Proyecto que abarca frontend, backend, base de datos y despliegue
- **MVP (Minimum Viable Product)**: Versión funcional mínima con las características esenciales
- **Project Scaffolding**: Estructura inicial del proyecto con configuración base
- **Architecture Decision Record (ADR)**: Documentación de decisiones arquitectónicas clave
- **Monorepo**: Repositorio único que contiene múltiples proyectos relacionados
- **Demo Application**: Aplicación funcional desplegada que muestra características del proyecto
- **Reference Implementation**: Implementación de referencia que sigue las mejores prácticas
- **Seed Data**: Datos de ejemplo para poblar la base de datos y demostrar funcionalidad
- **Environment Configuration**: Gestión de configuración por entorno (dev, staging, production)
- **CI/CD Pipeline**: Pipeline automatizado de integración y despliegue continuo

## Tecnologías principales

| Herramienta | Propósito |
|-------------|-----------|
| Next.js / Remix | Full-stack React con SSR y API routes |
| Django + DRF | Backend Python completo con ORM y API REST |
| Spring Boot | Backend Java enterprise con ecosistema completo |
| FastAPI | Backend Python moderno con OpenAPI automático |
| PostgreSQL / MongoDB | Bases de datos para proyectos full-stack |
| Docker Compose | Orquestación multi-servicio para desarrollo local |
| Vercel / Netlify | Despliegue serverless de frontends |
| Railway / Fly.io | Plataformas para desplegar backends completos |
| Supabase / Firebase | Backend as a Service con base de datos y auth |
| Turborepo / Nx | Monorepo tools para proyectos multi-paquete |

## Hoja de ruta

**Nivel Principiante:**
1. Crear proyecto TODO app full-stack (React + Express + SQLite)
2. Desplegar portafolio personal con Next.js + Vercel
3. Construir API REST básica con FastAPI + PostgreSQL
4. Implementar CRUD completo con autenticación

**Nivel Intermedio:**
1. Desarrollar e-commerce con carrito, pagos y panel admin
2. Crear aplicación con WebSockets (chat en tiempo real)
3. Implementar CI/CD con GitHub Actions y despliegue automatizado
4. Construir app con autenticación OAuth2 y roles de usuario

**Nivel Avanzado:**
1. Diseñar sistema multi-tenant con aislamiento de datos
2. Implementar microfrontends + backend for frontend (BFF)
3. Desplegar arquitectura event-driven con Kafka/RabbitMQ
4. Crear plataforma SaaS completa con billing, monitoreo y escalado
