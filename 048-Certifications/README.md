# 048-Certifications: Certificaciones Técnicas

## Descripción del dominio

Las certificaciones técnicas validan formalmente los conocimientos y habilidades de profesionales en tecnologías específicas, plataformas cloud, metodologías y dominios especializados. Este módulo organiza información sobre las principales certificaciones de la industria, incluyendo rutas de preparación, temarios, recursos de estudio, costos, duración de las certificaciones y estrategias para aprobar los exámenes. Cubre certificaciones en cloud computing (AWS, Azure, GCP), contenedores y orquestación (Kubernetes, Docker), infraestructura como código (Terraform, Ansible), bases de datos, SAP, seguridad (CISSP, CompTIA Security+), gestión de proyectos (PMP, PRINCE2) y especialidades emergentes en IA/ML y DevOps.

## Conceptos clave

- **Certificaciones AWS**: Cloud Practitioner (fundational), Solutions Architect (Associate/Professional), Developer Associate, SysOps Administrator, DevOps Engineer, Security Specialty, Machine Learning Specialty, Advanced Networking, SAP on AWS
- **Certificaciones Microsoft Azure**: AZ-900 (Fundamentals), AZ-104 (Administrator), AZ-204 (Developer), AZ-305 (Architect), AZ-400 (DevOps), DP-100 (Data Scientist), SC-100 (Security)
- **Certificaciones Google Cloud**: Cloud Digital Leader, Associate Cloud Engineer, Professional Cloud Architect, Professional Data Engineer, Professional ML Engineer, Professional Security Engineer
- **Kubernetes (CNCF)**: CKA (Administrator), CKAD (Developer), CKS (Security), KCNA (Associate)
- **HashiCorp**: Terraform Associate (basics/003), Vault Associate, Consul Associate, Boundary Associate
- **DevOps/SRE**: GitLab Certified, Jenkins Certified, Docker DCA, Red Hat DO447 (Ansible), SRE Foundation (Google/DevOps Institute)
- **SAP**: SAP Certified Application Associate, SAP Activate, SAP S/4HANA, SAP BTP, SAP Fiori
- **Seguridad**: CISSP (ISC2), CompTIA Security+/CySA+/PenTest+, CEH (EC-Council), OSCP (Offensive Security), CISM (ISACA), CCSP (Cloud Security)
- **Gestión de proyectos**: PMP (PMI), PRINCE2 (Foundation/Practitioner), SAFe Agilist, Scrum Master (CSM, PSM I), ITIL 4 Foundation
- **Bases de datos**: Oracle DBA (OCP), MongoDB (C100DBA), MySQL (Oracle), PostgreSQL (EDB), Redis, Snowflake, Databricks
- **IA/ML**: AWS ML Specialty, GCP ML Engineer, Azure Data Scientist, TensorFlow Developer, Hugging Face + DeepLearning.AI, NVIDIA DLI
- **Estrategias de estudio**: Exam guides, practice tests (Whizlabs, TutorialsDojo, A Cloud Guru), dumps éticos, hands-on labs, comunidades (r/AWSCertifications, r/kubernetes)

## Hoja de ruta

1. **Principiante**: AWS Cloud Practitioner/AZ-900/GCP Cloud Digital Leader — CompTIA Security+ — Scrum Master (PSM I) — Terraform Associate
2. **Intermedio**: AWS SA Associate/AZ-104/GCP ACE — CKAD/CKA — PMP o PRINCE2 — AWS Developer Associate — ITIL 4
3. **Avanzado**: AWS SA Professional/AZ-305/GCP PCA — CKS — CISSP — AWS ML Specialty — Terraform 003
4. **Experto**: Múltiples certificaciones Professional/Specialty — OSCP — AWS DevOps Pro + SA Pro — SAP Certified Professional — arquitecto multi-cloud certificado — contribuciones a programas de beta exams
