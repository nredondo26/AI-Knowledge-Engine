# freeCodeCamp — Aprendizaje Práctico Gratuito

## ¿Qué es freeCodeCamp?

Organización sin fines de lucro fundada por Quincy Larson en 2014. Plataforma de aprendizaje de programación 100% gratuita con plan de estudios desde HTML hasta machine learning.

## Filosofía

- **100% gratuito** — sin costos ocultos
- **Aprender haciendo** — proyectos prácticos obligatorios
- **Comunidad global** — foros, YouTube, podcast
- **Open source** — contribuciones en GitHub

## Metodología

```text
Leer → Escribir código → Pasar tests → Proyecto → Certificación
```

Cada certificación (~300 horas) incluye:
- Lecciones teóricas interactivas
- Ejercicios con tests automáticos
- Proyectos finales integradores
- Certificado descargable + LinkedIn

## Editor Integrado

- Resaltado de sintaxis
- Consola y tests automáticos
- Sistema de pistas (hints)
- Reset de código

## Estrategias de Estudio

1. Consistencia: 1-2 horas diarias
2. Modificar proyectos con funcionalidades propias
3. Encontrar compañeros en el foro
4. Enseñar conceptos a otros (blog, videos)
5. Intentar resolver antes de ver hints
6. Repetir certificaciones para consolidar
