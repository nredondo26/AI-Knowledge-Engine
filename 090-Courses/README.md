# 090-Courses — Cursos Online

## Descripción del dominio

Directorio que cataloga, organiza y recomienda cursos online de las principales plataformas educativas. Cubre desde fundamentos de programación hasta especializaciones avanzadas en IA, cloud, DevOps, ciberseguridad y más. Cada entrada incluye nivel, duración, plataforma, instructor y valoración técnica para facilitar la selección del curso adecuado según el perfil del estudiante.

## Conceptos clave

- **MOOC** (Massive Open Online Courses): Cursos masivos abiertos en línea con miles de participantes
- **Especialización**: Conjunto de cursos que cubren un área completa (ej. Deep Learning Specialization)
- **Certificado profesional**: Credencial emitida por la plataforma al completar un programa
- **Hands-on**: Enfoque práctico con laboratorios, proyectos y ejercicios interactivos
- **Peer review**: Evaluación entre pares para proyectos y tareas
- **Instructor-led vs Self-paced**: Formato guiado por instructor vs ritmo propio
- **Sandbox**: Entorno preconfigurado para prácticas sin configuración local
- **Capstone**: Proyecto final integrador que consolida todo lo aprendido
- **Continuing Education Units (CEU)**: Créditos para educación continua profesional
