## Cómo navegar

1. **Empieza aquí** — vista completa del conocimiento
2. **Por tema** — elige el directorio del área que te interesa
3. **Profundiza** — cada README.md tiene conceptos, tecnologías y roadmap
4. **Conecta** — sigue `../XXX-Name/` para explorar módulos relacionados
5. **Aprende** — `090-Courses/`, `050-LearningPaths/`
6. **Consulta** — `091-CheatSheets/`, `092-FAQ/`, `093-CommonErrors/`
7. **Opera** — `095-Performance/`, `096-Optimization/`, `097-Observability/`
